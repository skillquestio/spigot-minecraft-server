# Spigot Minecraft Server

## Installing the server files

Windows:

```bash
.\install.bat
```

macOS:

```bash
chmod +x ./install.bash
./install.bash
```

Ubuntu:

```bash
chmod +x ./install.sh
./install.sh
```

## Starting the server

Windows:

```bash
.\start.bat
```

macOS and Ubuntu:

```bash
chmod +x ./start.sh
./start.sh
```
